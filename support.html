Pertanyaan yang Sering Ditanyakan
Ya, Hafia gratis untuk digunakan. Versi gratis didukung oleh iklan yang tidak mengganggu. Tersedia juga Hafia Premium — pembelian satu kali (tanpa langganan) untuk menghapus iklan selamanya.
Semua data transaksi disimpan secara lokal di perangkat Anda menggunakan SQLite. Hafia tidak mengunggah data keuangan Anda ke server manapun. Privasi Anda terjamin penuh. Anda dapat membackup data secara manual ke file .hafia atau ke Google Drive melalui menu Pengaturan → Backup.
Buka Pengaturan → Backup & Restore. Tersedia dua opsi: (1) Backup lokal — file .hafia disimpan di perangkat Anda dan dapat dibagikan via WhatsApp/email. (2) Google Drive — masuk dengan akun Google dan upload backup ke folder privat aplikasi. Backup otomatis juga dapat dijadwalkan setiap 1, 3, 7, 14, atau 30 hari.
Fitur Scan AI menggunakan Google Gemini API. Untuk menggunakannya: buka Pengaturan → AI Assistant → Google Gemini, masukkan API Key Gemini Anda (dapat diperoleh gratis di aistudio.google.com). Setelah itu, tombol scan di halaman utama akan aktif dan Anda bisa memfoto struk untuk mencatat otomatis.
Di halaman Grafik, ketuk tombol ekspor di pojok kanan atas. Pilih format yang diinginkan: PDF (laporan siap cetak), Excel (.xlsx), atau CSV. Data juga dapat diekspor sebagai file .hafia untuk keperluan backup atau migrasi ke perangkat lain.
Buka Pengaturan → Keamanan → Atur PIN. Setelah PIN diatur, Anda dapat mengaktifkan Face ID atau Touch ID sebagai alternatif yang lebih cepat. Tersedia juga kode darurat 8 digit untuk pemulihan jika lupa PIN.
Di layar PIN, ketuk "Lupa PIN" lalu pilih "Kode Darurat". Masukkan kode darurat 8 digit yang ditampilkan saat Anda pertama kali mengatur PIN — kode ini memungkinkan reset PIN tanpa kehilangan data. Setelah kode darurat digunakan, kode baru akan digenerate secara otomatis. Pastikan selalu menyimpan kode darurat di tempat yang aman (misalnya di aplikasi catatan terenkripsi).
Buka Pengaturan → Notifikasi, aktifkan Pengingat Harian, lalu atur jam yang diinginkan. Pastikan izin notifikasi sudah diberikan ke aplikasi di pengaturan perangkat Anda.
Ekspor data dari perangkat lama melalui Pengaturan → Backup → Backup Sekarang. Kirim file .hafia ke perangkat baru (via WhatsApp, email, AirDrop, dll). Di perangkat baru, buka file .hafia tersebut — Hafia akan otomatis terbuka dan menampilkan pratinjau sebelum mengimpor data.
Ya. Buka menu Kategori dari navigasi bawah, ketuk tombol tambah untuk membuat kategori baru dengan nama dan ikon pilihan Anda. Urutan kategori juga dapat diubah dengan drag & drop.
Hafia Premium adalah pembelian satu kali (tanpa biaya berulang) untuk menghapus semua iklan selamanya. Buka Pengaturan → Hapus Iklan untuk melakukan pembelian. Semua fitur aplikasi tetap tersedia di versi gratis.
Hafia tersedia untuk iOS (iPhone) dan Android. Aplikasi dioptimalkan untuk smartphone. Penggunaan di tablet memungkinkan namun belum dioptimalkan sepenuhnya.
